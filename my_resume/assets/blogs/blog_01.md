## Description 
Many times in a month, I hear: "Why haven’t you logged your daily task?"
Sometimes I work a lot but still get scolded just because I forgot to log it.

In my current company, we use Google Sheets for daily task logging. So I thought—why not make a shortcut? What if my program does the logging, and I just focus on coding?

This works well for me because most of my work involves development, and my GitHub commits reflect what I’ve done. So I built a small tool that fetches my commits and updates the Google Sheet automatically using the Google Sheets API. Now, I don’t have to think about logging—my code does it for me.

It’s a small thing, but I enjoyed building it. Solving my own problem and improving my daily workflow through code feels great.
